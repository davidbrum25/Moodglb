# Moodglb
GLB World loader.

A sleek, lightweight web-based 3D viewer for `.glb` and `.gltf` files built with Three.js. Features a beautiful dark UI and FPS-style flying controls, designed to quickly preview 3D assets in a clean, distraction-free environment.

## Features
- **Drag & Drop Loading**: Quickly import `.glb` or `.gltf` files by dropping them into the UI or clicking to browse.
- **Outliner & Transform Controls**: Manage multiple loaded models. Adjust Location, Rotation, and Scale for each model directly from the UI.
- **Animation Support**: Automatically plays embedded animations, with toggles to pause/resume individual model animations.
- **FPS Flight Camera**: Smooth, WASD-based flying controls with PointerLock for immersive navigation.
- **Customizable Experience**: 
  - Adjust flight speed.
  - Adjust camera focal length (FOV).
  - Toggle environment lights.
  - Toggle shadows.
  - Toggle grid and axes helpers.

## Hotkeys

### Movement
- **`WASD` / `Arrow Keys`**: Move Horizontally.
- **`Space` / `Ctrl`**: Move Up / Down (Elevation).
- **`Shift`**: Sprint (Move faster).

### Controls & Toggles
- **`ESC`**: Unlock cursor and open the sidebar menu.
- **`Shift + Mouse Wheel`**: Adjust Camera Focal Length (Zoom out/in).
- **`C`**: Toggle Environment Lights.
- **`X`**: Toggle Shadows.
- **`Z`**: Toggle Grid & Axes Helpers.
